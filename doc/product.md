# PRODUCT

Name:
Samarkand Mosaic

Type:
Telegram Mini App + Web Game

Genre:
Relaxing Puzzle + ASMR + Restoration

Core Idea:
Player solves mosaic puzzles inspired by Samarkand architecture.

Main emotional goal:
Player must feel:

"I am restoring ancient royal mosaics."

This is not just puzzle.
This is cultural restoration.

Core Loop:

Solve Puzzle
→ Snap Pieces
→ Feel Satisfaction
→ Complete Board
→ Relax
→ Next Level

Main Pillars:

1. Satisfaction
2. Beauty
3. Relaxation
4. ASMR
5. Timurid culture

Target Audience:

* casual players
* relaxing game lovers
* ASMR lovers
* puzzle players
* Telegram users

Main Product Rule:

Every interaction must feel premium.
