# GAMEPLAY

Core:

Drag piece
Drop piece
Snap piece
Complete board

Rules:

1. Piece must follow cursor exactly.
2. No offset allowed.
3. No lag allowed.
4. Snap must feel magnetic.
5. Wrong placement returns piece.

Hearts:

Start = 3

Wrong placement:
-1 heart

Game Over:
when hearts == 0

Levels:

L1 = 4 pieces
L2 = 6 pieces
L3 = 8 pieces
L4 = 10 pieces
L5 = 12 pieces

Snap Rules:

Snap radius = 80px

Correct:
auto snap

Wrong:
shake + return

Win:

All pieces placed.

Progress bar:

completed / total
